HTML(Hyper Text Markup Language)

=tag를 사용해 다양한 정보를 표현한다.

기본적 원리는 code(language)라는 언어를 Editor에 입력하여 컴퓨터에 출력한다. principle = '<>'와 그 안에 tag를 입력
- 가령, 진하게 표시 는 tag 'strong,/strong'를 사용
- 중요한 word에 밑줄을 치기위해 tag 'u,/u'를 사용 (mean, underline)
- 글꼴 크기를 조정하기 위해, tag 'h1, h2, h3, h4, h5, h6'
- 문장간에 간격을 띄우기 위해 tag 'br'을 사용 (mean, line break)
- 문단간에 간격을 띄우기 위해(=의미있는 정보를 나눠서 표시) tag 'P,/P'를 사용 (mean, paragraph)



- 이미지를 넣기 위해 TAG "img"와 Attribute(속성) "src=이미지.jpg" 사용 (mean, image, source)
- 목차(list)만들기, tag "li,/li" 와 부모tag "ul,/ul" 혹은 "ol,/ol" 사용 (mean, unordered list and ordered list) unordered=숫자를 매기지 않음, ordered=자동으로 list에 숫자를 매김
- [head] web page가 작업창에 표시되는 제목을 표시하기 위해 tag "title,/title" 사용빈도 높음
- [head] broswer에서 원하는 형식으로 page를 읽게하기 tag "meta charset=" (mean, character(문자) set(규칙))

*Absolute 본문에 표시되지 않는 부분을 묶는 것이 tag = "head,/head", 본문에 표시되어 확인할 수 있는 부분을 묶는 것이 tag = "body,/body", 이 모든 것을 묶는 단 하나의 tag = "html,/html"

- 형식적으로 가장 위에 !doctype html 표시
- 정보 전달의 근본 tag "anchor = 닻을 내리다.(=link)" "a href="",/a" hypertext 즉, 정보의 연결을 의미한다. (a means "anchor" ,anchor means "hypertext", hypertext means link, ref means reference=참조)
